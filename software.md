I (co-)developed a number of R-packages:

**VARcheck**
Assessing the model fit of Vector Autoregressive (VAR) models.  
[![CRAN status](https://www.r-pkg.org/badges/version/VARcheck)](https://CRAN.R-project.org/package=VARcheck) [![CRAN downloads](https://cranlogs.r-pkg.org/badges/last-month/VARcheck)](https://CRAN.R-project.org/package=VARcheck) [![GitHub](https://img.shields.io/badge/GitHub-repo-black?logo=github)](https://github.com/bsiepe/VARcheck)

**ClusterVAR**
Fitting Latent Class Vector Autoregressive (VAR) models.  
[![CRAN status](https://www.r-pkg.org/badges/version/ClusterVAR)](https://CRAN.R-project.org/package=ClusterVAR) [![CRAN downloads](https://cranlogs.r-pkg.org/badges/last-month/ClusterVAR)](https://CRAN.R-project.org/package=ClusterVAR) [![GitHub](https://img.shields.io/badge/GitHub-repo-black?logo=github)](https://github.com/aniebee/ClusterVAR)

**mnet**
A toolbox for modeling manifest and latent group differences and moderation effects in various statistical network models.  
[![CRAN status](https://www.r-pkg.org/badges/version/mnet)](https://CRAN.R-project.org/package=mnet) [![CRAN downloads](https://cranlogs.r-pkg.org/badges/last-month/mnet)](https://CRAN.R-project.org/package=mnet) [![GitHub](https://img.shields.io/badge/GitHub-repo-black?logo=github)](https://github.com/jmbh/mnet)

**mgm**
Estimation of Time-Varying Mixed Graphical Models.  
[![CRAN status](https://www.r-pkg.org/badges/version/mgm)](https://CRAN.R-project.org/package=mgm) [![CRAN downloads](https://cranlogs.r-pkg.org/badges/last-month/mgm)](https://CRAN.R-project.org/package=mgm) [![GitHub](https://img.shields.io/badge/GitHub-repo-black?logo=github)](https://github.com/jmbh/mgm)

**inet**
Performs inference with the lasso in Gaussian Graphical Models.  
[![CRAN status](https://www.r-pkg.org/badges/version/inet)](https://CRAN.R-project.org/package=inet) [![CRAN downloads](https://cranlogs.r-pkg.org/badges/last-month/inet)](https://CRAN.R-project.org/package=inet) [![GitHub](https://img.shields.io/badge/GitHub-repo-black?logo=github)](https://github.com/jmbh/inet)

**mousetrap**
Functions for importing, preprocessing, analyzing, aggregating, and visualizing mouse-tracking data.  
[![CRAN status](https://www.r-pkg.org/badges/version/mousetrap)](https://CRAN.R-project.org/package=mousetrap) [![CRAN downloads](https://cranlogs.r-pkg.org/badges/last-month/mousetrap)](https://CRAN.R-project.org/package=mousetrap) [![GitHub](https://img.shields.io/badge/GitHub-repo-black?logo=github)](https://github.com/PascalKieslich/mousetrap)

**fspe**
Estimating the number of factors in Exploratory Factor Analysis (EFA) via out-of-sampe prediction errors.  
[![CRAN status](https://www.r-pkg.org/badges/version/fspe)](https://CRAN.R-project.org/package=fspe) [![CRAN downloads](https://cranlogs.r-pkg.org/badges/last-month/fspe)](https://CRAN.R-project.org/package=fspe) [![GitHub](https://img.shields.io/badge/GitHub-repo-black?logo=github)](https://github.com/jmbh/fspe)

**cstab:** Selection of Number of Clusters via Resampled Normalized Cluster Stability.  
[![CRAN status](https://www.r-pkg.org/badges/version/cstab)](https://CRAN.R-project.org/package=cstab) [![CRAN downloads](https://cranlogs.r-pkg.org/badges/last-month/cstab)](https://CRAN.R-project.org/package=cstab) [![GitHub](https://img.shields.io/badge/GitHub-repo-black?logo=github)](https://github.com/jmbh/cstab)
